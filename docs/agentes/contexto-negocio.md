# Contexto De Negocio

El proyecto gestiona procesos vinculados a materiales, SAT, calendario, planos, presupuestos, roles, técnicos, gestores y archivos de trabajo como Excel, OneDrive y PDFs.

## Conceptos Clave

- Materiales: insumos o elementos necesarios para trabajos, obras o mantenimiento.
- SAT: flujo de asistencia o servicio técnico.
- Calendario: planificación de visitas, tareas o eventos operativos.
- Planos: documentación técnica sensible asociada a trabajos o clientes.
- Presupuestos: información económica que debe tratarse con cuidado.
- Roles: permisos y responsabilidades de cada tipo de persona usuaria.
- Técnicos: personas que ejecutan tareas operativas.
- Gestores: personas que coordinan, revisan o administran información.

## Cuidado Con Datos

Excel, OneDrive y PDFs pueden contener datos reales. Hay que tratarlos como carpetas físicas de una oficina: antes de mover, borrar o publicar algo, se revisa qué contiene y quién debería verlo.
