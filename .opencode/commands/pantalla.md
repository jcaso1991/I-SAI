---
description: Crear o ajustar pantallas de frontend con criterio mobile-first y accesible.
---

Si no hay pedido después de `/pantalla`, preguntá: "¿Qué pantalla querés crear o mejorar?" y terminá el turno.

Aplicá las skills `frontend-expo-industrial`, `comunicacion-usuario` y `qa-seguro`.

Flujo:

1. Revisá la estructura existente de frontend antes de tocar.
2. Trabajá mobile-first (primero celular, después pantallas grandes).
3. Respetá accesibilidad (que se pueda usar con lector de pantalla, buen contraste y controles claros).
4. Mantené el diseño industrial del proyecto.
5. No corras builds.
6. Verificá con revisión estática o comandos seguros.
