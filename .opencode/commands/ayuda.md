---
description: Mostrar comandos disponibles y revisar acceso a GitHub sin exponer salida técnica.
---

Primero ejecutá `gh auth status`.

Si pasa, no muestres salida técnica.

Si falla, mostrale a la persona:

"Tu acceso a GitHub expiró. Ejecutá `gh auth login` o pedile ayuda a quien mantenga el proyecto."

Después listá comandos con ejemplos coloquiales:

- `/mejorar hacé más claro el calendario`
- `/arreglar no guarda el presupuesto`
- `/pantalla mejorar la vista de técnicos en celular`
- `/datos revisar permisos de presupuestos`
- `/verificar backend`
- `/revisar`
- `/guardar`
- `/deshacer`
- `/pedir-ayuda no puedo resolver este error`
- `/explicar este error`

Usá español claro y breve.
